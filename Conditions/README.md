# Conditions

 Eğer bir olayın gerçekleşmesi için bir koşul, şart varsa iki switch yada if...else metodları kullanılır.

* if...else<br>
```c#
if (condition)
{
  // block of code to be executed if the condition is True
}
```
* switch<br>
```c#
switch(expression)
{
  case x:
    // code block
    break;
  case y:
    // code block
    break;
  default:
    // code block
    break;
}
```
